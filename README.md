# DSA & System Design Playground

A fully interactive, static website to master Data Structures, Algorithms, LeetCode patterns, and System Design for interviews. Includes:
- Step-by-step problem solving approach
- 14 LeetCode patterns (with code, quizzes, and explanations)
- Data structures and core algorithms
- Sorting visualizer (animated)
- System design concepts and case studies
- Progress tracking (localStorage)

## Demo
Open `index.html` in your browser, or run locally:

```sh
python3 -m http.server 8080
```
Then visit [http://localhost:8080](http://localhost:8080)

## How to Use
- Click through sidebar or hero cards to explore topics
- Mark topics as learned to track progress
- Use the sorting visualizer to see algorithms in action
- Take quizzes to reinforce learning
